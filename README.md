# Gauge Design System

A modern design system built with React and TypeScript, implementing components from Figma designs.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Components

- **Button** - A versatile button component with multiple variants and states

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS Modules
